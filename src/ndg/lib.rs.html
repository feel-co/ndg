1//! # NDG: Not a Docs Generator
2//!
3//! `ndg` is a fast, customizable documentation generator for Nix, `NixOS`, and
4//! Nixpkgs module systems. It converts Markdown and Nix module options into
5//! HTML and manpages, supporting Nixpkgs-flavored `CommonMark`, automatic table
6//! of contents, search, multi-threading, and fully customizable templates.
7//!
8//! ## Features
9//! - Markdown to HTML and Manpage conversion with Nixpkgs-flavored `CommonMark`
10//!   support
11//! - Automatic table of contents and heading anchors
12//! - Search functionality across documents
13//! - Nix module options support for generating documentation from
14//!   `options.json`
15//! - Fully customizable templates and stylesheets
16//! - Multi-threading for fast generation of large documentation sets
17//! - Seamless integration with Nix workflows
18//!
19//! ## Usage
20//!
21//! Be advised that NDG's internal API is exposed ONLY in order to be used in
22//! our unit tests. While this API could be useful to end users, e.g., by
23//! exposing some wrapper functions, [ndg-commonmark](../ndg_commonmark)'s own
24//! API should be preferred for library use. This interface is for
25//! testing purposes ONLY and will not make any guarantees of stability between
26//! versions.
27//!
28//! ### CLI Use
29//!
30//! [Github Repository]: https://github.com/feel-co/ndg
31//! NDG is primarily designed as a CLI utility, and its documentation is located
32//! in the [Github repository]. Please refer to the project README for more
33//! information about the project, installing NDG, and CLI usage.

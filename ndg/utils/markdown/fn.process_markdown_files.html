pub fn process_markdown_files(config: &Config) -> Result<Vec<PathBuf>>
Expand description

Processes all markdown files in the input directory and writes HTML output.

This function renders all markdown files, handles custom output paths, and writes the resulting HTML files to the output directory. Files that are included in other files are skipped unless they have custom output paths.

§Arguments

  • config - The loaded configuration for documentation generation.

§Returns,

A vector of all processed markdown file paths.

§Errors

Function sanitize_option_id_legacy 

Source 
fn sanitize_option_id_legacy(name: &str) -> String
Expand description

Produce a legacy HTML id / URL fragment from an option name.

This function produces IDs using the old format where dots are replaced with hyphens, for compatibility with older documentation builds.

§Arguments

  • name - The option name to sanitize

§Returns

A String suitable for use as an HTML id attribute, prefixed with “option-” and with special characters (but not dots) replaced with underscores.

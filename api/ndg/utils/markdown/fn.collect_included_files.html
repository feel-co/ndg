Expand description

Collects all included files from markdown documents in the input directory.

This scans all markdown files and collects the set of files that are included via NDG’s markdown processor.

§Arguments

  • config - The loaded configuration for documentation generation.
  • processor - Optional pre-created processor to reuse. If None, creates a new one.

§Returns

A mapping of included file paths (relative to input directory) to the parent input file path, which included that file

§Errors

Returns an error if any markdown file cannot be read.
